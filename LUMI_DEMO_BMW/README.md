# Lumi 义卖机器人 - 智能售货系统

## 项目简介

这是一个基于 **JAKA Lumi 协作机器人** 开发的智能售货系统，成功应用于宝马南京信息技术有限公司内部义卖活动现场。系统以 **智能体（Agent）** 作为核心大脑，通过自然语言与用户交互，调用 JAKA SDK 精确控制机器人完成饮料售卖全流程。

## 系统架构

系统采用"**智能体调度 + 机器人执行**"的双层架构：

### 智能体系统（上层）
- 理解用户的语音指令（如"我要一瓶可乐"）
- 解析用户意图并生成任务计划
- 监控整个售卖流程的状态
- 处理异常情况并做出决策

### 机器人控制系统（下层）
- 通过 JAKA SDK 控制 Lumi 机器人的精准运动
- 操作电动夹爪抓取和放置饮料瓶
- 接收传感器数据确保操作安全
- 与支付终端、显示屏等外围设备协同工作

---

## 工作流程

1. **用户唤醒**：当用户靠近时，机器人通过视觉检测自动唤醒，LED灯闪烁并语音问候
2. **商品选择**：用户通过语音说出想要的饮料（如"可乐"、"雪碧"），智能体解析指令并确认商品库存
3. **支付确认**：系统语音报价并显示支付二维码，用户扫码完成后，支付终端返回成功信号
4. **机器人执行**：智能体生成动作序列，机器人依次执行：
   - 从待机位置移动到饮料存放区
   - 精准定位目标饮料瓶
   - 使用夹爪稳定抓取
   - 平稳移动到用户取货窗口
   - 安全放置饮料并返回待机位置
5. **服务完成**：机器人语音提示用户取货，系统更新库存并记录销售数据

## 环境配置

### 1. 依赖安装
```bash
pip install -r requirements.txt
```

### 2. JAKA 机器人 SDK
将 `LumiAgent/JAKA_SDK_LINUX` 路径加入环境变量：
```bash
export LD_LIBRARY_PATH=/path/to/LUMI_DEMO_BMW/LumiAgent/JAKA_SDK_LINUX:$LD_LIBRARY_PATH
```

### 3. Orbbec 相机 SDK
推荐使用 [pyorbbecsdk](https://github.com/orbbec/pyorbbecsdk)，确保相机驱动和 SDK 已正确安装。
